# ViaSOS.it

Sito principale ViaSOS, piattaforma italiana per inviare una richiesta di soccorso stradale tramite posizione e WhatsApp.

## Variabili ambiente

Configura:

```bash
NEXT_PUBLIC_WHATSAPP_NUMBER=39XXXXXXXXXX
```

## Comandi

```bash
npm run dev
npm run lint
npm run typecheck
npm run build
```
